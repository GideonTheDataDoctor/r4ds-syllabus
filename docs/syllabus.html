üéØ LEARNING GOALS

By the end of this course, participants will be able to:

  1. Import & wrangle data ‚Äì Load, clean, reshape messy datasets into analysis-ready format

  3. Visualize effectively ‚Äì Create exploratory and publication-ready plots with ggplot2
  4. Write functions & automate ‚Äì Build reusable functions; iterate efficiently with purrr
  5. Model data ‚Äì Fit simple and complex models to understand relationships
  6. Communicate results ‚Äì Generate reproducible reports with R Markdown
  7. Apply to epidemiology ‚Äì Use all skills on real STH surveillance and research data

üìñ BOOK ORDER vs. OUR LEARNING SEQUENCE

Why We‚Äôre Reordering the Chapters

The R for Data Science book is published in this order: Visualization (Ch 1) ‚Üí Transformation (Ch 3) ‚Üí Wrangling (Ch 7‚Äì13) ‚Üí Programming (Ch 14‚Äì17) ‚Üí Modeling (Ch 18‚Äì20) ‚Üí Communication (Ch 21‚Äì24).

However, this order isn‚Äôt optimal for beginners learning epidemiology. The book‚Äôs authors acknowledge this in their Preface:

‚ÄúStarting with data ingest and tidying is suboptimal because 80% of the time it‚Äôs routine and boring, and the other 20% of the time it‚Äôs weird and frustrating. That‚Äôs a bad place to start learning a new subject! Instead, we‚Äôll start with visualization and transformation of data that‚Äôs already been imported and tidied.‚Äù

The problem: Chapter 1 (ggplot2) assumes clean data exists. But beginners need to import, clean, and reshape data FIRST before they can visualize or analyze it.

Our Reordered Learning Sequence

We follow the natural real-world epidemiological workflow instead of the book‚Äôs ordering:

Phase Weeks Our.Order Book.Chapters Purpose
1. Foundation 1‚Äì3 First 2, 4, 6 (Workflow) Learn RStudio, organize work, set up projects BEFORE coding
2. Wrangling 4‚Äì13 Second 7‚Äì13 (Import, Tidy, Transform, Join, Handle types) Get messy data into analysis-ready format
3. Exploration 14‚Äì15 Third 5, 1 (EDA, Visualization) Explore clean data visually & statistically
4. Programming 16‚Äì18 Fourth 15, 16, 17 (Functions, Vectors, Iteration) Write reusable code & automate workflows
5. Modeling 19‚Äì21 Fifth 18, 19, 20 (Model Basics, Building, Many Models) Fit & evaluate statistical models
6. Communication 22‚Äì25 Sixth 21‚Äì24 (R Markdown, Graphics, Formats, Workflow) Document findings & create reproducible reports

Why This Order Works Better

‚úÖ Motivation: Learn foundational tools first, then apply them to real problems
‚úÖ Real-world: Follow the actual epidemiological analysis workflow
‚úÖ Practical: Work with messy data early; clean it before analyzing
‚úÖ Confidence: See tangible results at each step (import ‚Üí clean ‚Üí analyze ‚Üí visualize)
‚úÖ Less frustration: Don‚Äôt start with ‚Äúboring‚Äù wrangling; build to it naturally
